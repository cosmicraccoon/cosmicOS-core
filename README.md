# cosmicOS-core

This is a simple to use HAL for ARM Cortex-M based MCUs  
IT'S A **WORK IN PROGRESS** SO STAY TUNED
More coming soon...

## Supported Platforms

- [x] STM32F4xx

## Features

- [x] GPIO
- [x] EXTI interrupts
- [x] USART
- [ ] I2C (work in progress)
- [ ] SPI
- [ ] DMA
- [ ] USB-OTG
- [ ] ADC
- [ ] SMBus
- [ ] Timers
more features coming soon...
